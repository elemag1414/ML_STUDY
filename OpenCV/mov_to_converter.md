# MOV to AVI Converter

- [Mov to AVI Converter 코드](mov_to_avi_converter.py)

```bash
$ python mov_to_avi_converter.py --input input_video.MOV --output output_video.AVI
```

--output 인자가 없으면 입력 비디오만 플레이 해줌.
